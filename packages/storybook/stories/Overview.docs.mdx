# FootyViz component library

> Data-first football companion. The UI disappears. The visualizations speak.

Monochrome-dominant design system with a single straw-yellow accent
(`#E8BE56`), Geist + Geist Mono, and dark/light at equal priority. Built as
vanilla web components — custom elements + light DOM, no React, no
Lit, no template engine. This page is the table of contents for everything
in the sidebar.

## Locker Room

Composed arrangements of components — the next layer up from primitives,

players that becomes greater than its parts). Shipped from
